将IntelliJ Idea的项目发布到github

Intelli Idea版本：

安装git并在Idea中设置好git的路径

在Idea设置-Version Control-github里登录github账号

创建git仓库

目录选择项目所在目录就可以

将修改提交到git仓库

在项目根目录处右键git-Add或使用快捷键Ctrl+Alt+A

Commit

将项目发布到github

不出意外的话，登录github就可以看到成果了。
