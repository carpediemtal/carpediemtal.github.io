Code segment

为什么代码块和数据块要分开？
清晰。一会数据一会代码不便于阅读。
代码块和数据块对于操作偶读需求不一样，分开便于权限管理。
代码去冗余，节约空间。对于程序中一段需要反复处理数据的函数，若数据与指令混合装载至内存，每次数据的不同必然导致内存的浪费。
