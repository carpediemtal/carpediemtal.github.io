Objectзұ»зҡ„ж–№жі•

Posted on 2020-10-20 Edited on 2025-02-13 In Java

equalsе’ҢhashCode

equals

й»ҳи®ӨдҪҝз”Ё==жҜ”иҫғеҜ№иұЎзҡ„еј•з”ЁпјҢд№ҹе°ұжҳҜиҜҙеҰӮжһңдёҚеҺ»иҰҶеҶҷпјҢдҪҝз”Ёequalsе’Ң==ж•ҲжһңжҳҜдёҖж ·зҡ„гҖӮ

hashCode

hashCode()жҳҜдёҖдёӘnativeж–№жі•пјҢеҖјз”ұеҜ№иұЎзҡ„еӯҳеӮЁең°еқҖи®Ўз®—еҫ—еҮәпјҲеӯ—з¬ҰдёІзұ»еһӢзҡ„hashеҖјж №жҚ®еӯ—з¬ҰдёІеҶ…е®№и®Ўз®—пјүгҖӮ

дёәд»Җд№ҲйңҖиҰҒиҰҶеҶҷequalsе’ҢhashCodeж–№жі•

дёӨдёӘж–№жі•зҡ„дҪңз”ЁйғҪжҳҜжҜ”иҫғдёӨдёӘеҜ№иұЎеңЁйҖ»иҫ‘дёҠжҳҜеҗҰзӣёзӯүгҖӮequalsеҸҜйқ пјҢиҖҢhashCodeжҖ§иғҪжӣҙејәгҖӮ

Objectй»ҳи®Өзҡ„equalsе’ҢhashCodeж–№жі•еҹәжң¬жІЎд»Җд№Ҳз”ЁгҖӮ

жүҖд»ҘеҰӮжһңиҰҒеҒҡеӨ§йҮҸзҡ„еҜ№жҜ”пјҢеҸҜд»Ҙе…Ҳз”ЁhashCodeеҒҡз¬¬дёҖжӯҘзӯӣйҖүпјҢеҰӮжһңhashCodeзӣёеҗҢеҶҚз”ЁequalsеҒҡеҮәжңҖеҗҺзҡ„еҲӨж–ӯгҖӮ

  • жғ…жҷҜдёҖпјҡжҲ‘жңүдёҖдёӘListпјҢе…ғзҙ зұ»еһӢжҳҜStudentпјҲListпјүпјҢжҲ‘жғіеҜ№Listдёӯзҡ„е…ғзҙ еҺ»йҮҚпјҢеҸҜд»ҘдҪҝз”ЁList.stream().distinct()ж–№жі•пјҢдҪҶжҳҜйңҖиҰҒе‘ҠиҜүjavaжҖҺд№Ҳж ·зҡ„дёӨдёӘStudentжүҚиғҪз§°д№ӢдёәequalгҖӮ
  • жғ…жҷҜдәҢпјҡHashMapдёӯж·»еҠ иҮӘе®ҡд№үзұ»еһӢзҡ„keyж—¶пјҢйңҖиҰҒиҰҶеҶҷequalsе’ҢhashCodeж–№жі•пјҢеҺҹеӣ иҰҒд»ҺHashMapзҡ„еҺҹзҗҶиҜҙиө·гҖӮ

еҰӮдҪ•иҰҶеҶҷequalsе’ҢhashCodeж–№жі•

equalsж–№жі•еҲӨж–ӯзӣёзӯүпјҢhashCodeеҖјеҝ…йЎ»зӣёзӯүгҖӮhashCodeеҖјзӣёзӯүпјҢequalsж–№жі•еҸҜд»ҘдёҚжғізӯүгҖӮ

жүҖд»ҘhashCodeзӣёзӯүжҳҜequalsзҡ„еҝ…иҰҒдёҚе……еҲҶжқЎд»¶гҖӮ

иҰҶеҶҷequalsж—¶йңҖиҰҒиҖғиҷ‘иҮӘеҸҚжҖ§гҖҒеҜ№з§°жҖ§гҖҒдј йҖ’жҖ§гҖҒдёҖиҮҙжҖ§пјҲзҰ»ж•Јж•°еӯҰйҮҢе…ізі»зҡ„жҖ§иҙЁпјүгҖӮ

toString

System.out.println()ж—¶й»ҳи®Өзҡ„ж–№жі•пјҢй»ҳи®ӨдёәвҖқзұ»еҗҚ@hashCodeвҖқгҖӮ

clone

еҰӮжһңеҜ№иұЎзҡ„зұ»жІЎжңүе®һзҺ°CloneableжҺҘеҸЈпјҢи°ғз”Ёcloneж–№жі•е°ұдјҡжҠӣеҮәCloneNotSupportedExceptionгҖӮ
