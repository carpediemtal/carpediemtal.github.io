Virtual Memory

Posted on 2020-08-13 Edited on 2025-02-12 In OS

概念

  1. 每个程序有自己的地址空间，这些地址空间被分为很多拥有连续地址范围的页面
  2. 程序运行时，并非所有页面都要加载进内存。只有需要时，页面才被OS映射到物理内存
  3. 页面可以被交换（和磁盘）
  4. 程序里用到的是虚地址
